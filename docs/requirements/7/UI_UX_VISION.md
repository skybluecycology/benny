## Expanded Interaction Model

### 1. 🛸 Spatial Navigation (3D View)
*   **The Mesh**: Swarms are no longer flat nodes. They are 3D energy clusters (`Intake`, `Orchestrator`, `Synthesis`).
*   **Agent Orbits**: Individual sub-agents orbit their cluster cores. The speed of the orbit and the length of the energy trails represent their current processing load/thinking speed.
*   **Data Shards**: Input files and context are rendered as floating 3D "Data Shards." Clicking a shard expands it into a holographic preview.

### 2. 🕹️ God-Mode HUD (The Control Frame)
*   **Governance Log (Safety)**: A dedicated vertical panel on the left that streams "Bios-Green" for success and "Phosphor-Orange" for warnings/violations.
*   **Synaptic Stream (Thinking)**: A corner overlay that provides the raw `AER` (Agent Execution Record) trace. It uses hex IDs and monospace typography to give a "machine-thought" aesthetic.
*   **Emergency HALT**: A massive, holographic circular button with internal rotating rings. It provides an immediate kill-switch that sends an interrupt signal to the entire mesh.

### 3. 🧠 3D Knowledge Synthesis
*   **The Synoptic Web**: Visualize the emerging knowledge graph in 3D. As the swarm processes information, new synaptic connections (purple/cyan) are drawn between concepts in real-time.
*   **Interactive Nodes**: Click any concept in the 3D web to see the specific agent reasoning and source documents that led to its creation.

### 4. 🛒 Agent Marketplace (The Swarm Hub)
*   **Skill Configuration**: Select and configure the "Cognitive Skills" of your swarm agents using a sleek, card-based 3D grid.
*   **Stability Metrics**: Monitor the "Cognitive Stability" of each agent type before deploying it into the mesh.

### 5. ⏳ Temporal Auditing (The Timeline)
*   **The Scrubber**: A bottom-aligned timeline that tracks the history of the run.
*   **Time-Travel**: Scrubbing backwards dims the current 3D state and highlights past active nodes, allowing you to visually audit "why" a specific decision was made.


### 4. 🔗 Directive Injection (Command Bar)
*   **Universal Intent**: A floating command bar at the bottom. Instead of editing the graph manually, you "broadcast" a new intent (e.g., "Pivot development to React Native"). The **Planner** node will visually expand the new tasks in the 3D space in real-time.

---

> [!TIP]
> This UI will use **React Three Fiber** for the 3D layer and **Framer Motion** for the fluid HUD transitions.
